# 个人网站架构总览

## 项目信息

- **项目名称**: RollingTheRock 个人网站
- **项目类型**: 静态内容网站（博客 + 项目展示 + 视频）
- **目标用户**: 技术读者、潜在雇主、社区成员
- **设计风格**: Every.to + Notion 极简风格

## 技术栈

| 层级 | 技术 | 版本 |

## 页面清单

| 页面 | 路径 | 说明 |
|------|------|------|
| 首页 | `/` | 侧边导航 + Hero + 最新内容 |
| 博客列表 | `/blog` | 文章列表，支持分类筛选 |
| 博客详情 | `/blog/[slug]` | 文章详情，支持 Giscus 评论 |
| 分类页 | `/blog/categories/[category]` | 按分类筛选 |
| 标签页 | `/blog/tags/[tag]` | 按标签筛选 |
| 项目列表 | `/projects` | 项目卡片网格 |
| 视频列表 | `/videos` | 视频卡片网格 |
| 视频详情 | `/videos/[slug]` | 嵌入式播放器 |
| 关于 | `/about` | 个人介绍 |
| 404 | `/404` | 自定义 404 |
| RSS | `/rss.xml` | RSS Feed |

## 关键决策摘要

1. **ADR-001**: 使用 Astro + Tailwind CSS
2. **ADR-002**: 采用 SSG + Islands Architecture
3. **ADR-003**: 纯静态生成，无 SSR
4. **ADR-004**: Tailwind + CSS Variables 主题系统
5. **ADR-005**: Giscus + Buttondown + Vercel

# 博客列表页改造计划

## 概述

改造博客列表页，向首页的复古杂志风格靠拢。

---

## 当前状态分析

### 现有博客列表页结构 (`src/pages/blog/index.astro`)

- 双栏布局：主内容区 + 右侧边栏 (280px)
- 页眉：居中对齐，中文"博客"标题 + 描述文字
- 分类筛选：圆角胶囊按钮，7 个分类
- 文章卡片：横向布局，图片左 + 内容右

### 涉及文件

1. `src/pages/blog/index.astro` - 博客列表主页面
2. `src/components/BlogCard.astro` - 文章卡片组件
3. `src/components/Sidebar.astro` - 侧边栏组件（将删除）

---

## 改动详述

### 改动 1：重设计页眉

**从：**
- 居中大号中文标题"博客"
- 居中副标题
- 大面积上下留白

**改为：**
- 左对齐杂志感页眉
- 英文花体标题 "Blog" (Playfair Display italic, 36px)
- 中文副标题紧跟其下 (14px 灰色)
- 底部分割线

```
┌─────────────────────────────────────┐
│ Blog                       全部/思考/ │
│ 探索技术，分享思考的文章集合  技术/... │
│─────────────────────────────────────│
└─────────────────────────────────────┘
```

### 改动 2：去掉右侧边栏

- 删除整个右侧边栏
- 页面变为单栏全宽布局
- 主内容区占满页面宽度

### 改动 3：简化分类筛选栏

**从 7 个选项改为 5 个：**

| 显示名称 | 包含的原分类 |
|---------|------------|
| 全部 | 所有文章 |
| 思考 | 思考 |
| 技术 | 技术、ACM、基础算法 |
| 生活 | 生活、随笔 |
| 项目 | 项目 |

**样式改变：**
- 纯文字 `/` 分隔
- 选中项：品牌色 `#A855F7` + 下划线
- 未选中项：灰色 `#6B7280`

### 改动 4：文章卡片改造

**图片区域：**
- 默认灰度 (`filter: grayscale(100%)`)
- hover 去灰度 + 微放大

**标题区域：**
- 标题前加品牌色圆点
- hover 时标题变品牌色、圆点发光

**分类标签：**
- 去掉彩色背景胶囊
- 改为纯文字，品牌色

**卡片列表 hover 效果：**
- hover 某张卡片时其他卡片变暗 (opacity: 0.5)

### 分类颜色映射

```typescript
const categoryColorMap: Record<string, string> = {

```

## 响应式处理

### ≤960px
- 页眉：分类筛选从右侧移到标题下方
- 文章卡片：从左图右文横向排列变为上图下文纵向排列

### ≤480px
- 卡片全宽纵向排列

## 项目概述

从零搭建的个人博客系统。不只是一个技术项目，更是一次对"个人网站应该是什么样子"的深度探索。

每一个设计细节——从复古未来主义的配图风格到衬线体排版，从胶片颗粒感的视觉质感到三色品牌系统——都经过反复打磨，力求在技术实现与审美表达之间找到平衡。

## 设计理念

### 复古未来主义

网站的视觉风格定义为"复古未来主义"——古典素描笔触与未来科技意象的碰撞。所有配图采用统一的炭笔素描风格，搭配强烈的胶片颗粒感，营造出一种"从未来寄回的旧照片"的独特氛围。

### 三色品牌系统

建立了贯穿全站的色彩语言：

- **紫色 #7C3AED** — 思考与 AI：用于博客思考类文章和 AI 相关项目
- **青色 #0891B2** — 技术与工程：用于博客技术类文章和 Web 类项目
- **橙色 #EA580C** — 生活与创意：用于博客生活类文章和工具类项目

每个分类只使用一个品牌亮色搭配复古白/灰，确保视觉的克制与统一。

### 排版优先

以 Cormorant Garamond 衬线体作为标题字体，等宽字体作为辅助，在深色背景上营造高对比度的阅读体验。

## 核心特性

### Z 字形导航入口

首页 Blog / Projects / Videos 三个入口采用左右交替堆叠布局，配合 AI 生成的复古未来主义素描配图。每个入口都有独立的品牌色标识：hover 时品牌色光晕从图片扩散，分隔线变为对应品牌色，营造沉浸式浏览体验。

### 统一配图体系

建立了完整的配图风格规范和 Prompt 模板系统：

- 所有配图共享复古未来主义视觉基底
- 每个分类对应固定品牌色，图中只出现一个亮色
- 配图主体与内容之间是隐喻关系而非直白图解
- 统一的 Midjourney / 豆包 Prompt 模板确保风格一致性
- 标准化的后期处理流程

### 深色模式优先

以深色背景 (#0a0a0a) 为基础设计，所有元素（配图、排版、交互）都围绕深色模式优化，同时支持浅色模式切换。

### 响应式设计

从桌面端到移动端的完整适配，导航入口在移动端自动切换为垂直布局。

## 技术栈

- **框架**：Astro — 静态站点生成，极致性能
- **样式**：Tailwind CSS — 原子化 CSS，高效开发
- **语言**：TypeScript — 类型安全
- **部署**：Vercel / Netlify
- **订阅**：Buttondown 邮件订阅集成
- **配图**：AI 生成 (Midjourney / 豆包) + Photopea 后期处理
